# [백준2178] 미로탐색

- URL :  https://www.acmicpc.net/problem/2178
- 출처 :  X



## 개요

> 목표

(1, 1)부터 (N, M) 까지 걸리는 최소시간을 구하라



> 제약조건

- N, M(2 ≤ N, M ≤ 100)
- 항상 도착위치로 이동할 수 있는 경우만 입력으로 주어진다.




## 계획

1. 탐색 알고리즘을 사용하면 쉽게 풀 수 있는 문제라고 판단했습니다.
2. char형 배열에 map 정보를 저장하고 (0, 0)부터 (N - 1, M - 1)까지 도달할 때 걸리는 시간을 체크했습니다.



## 결과

걸린시간: 약 20분
